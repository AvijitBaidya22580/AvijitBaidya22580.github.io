Major Project: Hybrid Malware Detection

Goal: To develop a robust system capable of detecting zero-day and sophisticated evasive malware by integrating Reverse Engineering with Machine Learning[cite: 36, 37].

The Hybrid Approach

Traditional signature-based methods often fail against polymorphic malware. My system combines the structural depth of Static Analysis with the behavioral truth of Dynamic Analysis.

Technical Breakdown

  • Dynamic Analysis: Utilizing Cuckoo Sandbox to execute malware in isolated Virtual Machines and generate detailed JSON activity reports[cite: 101, 102].
  • Static Analysis: Using the pefile Python library to parse Portable Executable (PE) headers and extract metadata[cite: 104, 105].
  • ML Algorithm: Implementing a Random Forest ensemble learning method to build a robust classification model[cite: 108, 109].

System Performance

The proposed system aims for a significant performance leap over baseline methods[cite: 146, 153]:

  • Detection Rate (Recall): >95% (Baseline: 85%)[cite: 153].
  • F1-Score: >90% (Baseline: 80%)[cite: 157].
  • False Positive Rate: Approximately 1%[cite: 159].
