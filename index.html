Smart Grid Intrusions NIT Puducherry

Decentralized Smart Grid Intrusion Detection

Designed a Federated Graph Attention Network (GAT) model to secure power grid meters.

Problem:

Centralized monitoring of Smart Meters and IEDs exposes sensitive grid state data and creates single points of failure for state-sponsored attacks.

Solution:

Implemented a Federated Learning structure with GAT models, allowing local nodes to train anomaly classifiers on graph-structured grid topology without sharing raw state metrics.

Impact:

Maintained high detection accuracy on fault injection and spoofing attacks while ensuring data privacy across localized clients.
